# 1001 件人类艺术瑰宝

一个精选并展示人类历史上 1001 件著名艺术作品的静态网站，跨越史前壁画到当代艺术。

## 在线预览

https://xujiann.github.io/1001art/

## 功能

- **1800+ 件真实作品**，其中 **1790+ 件配有本地缓存的高清图片**（不再依赖外部链接）
- **按艺术家浏览**：艺术家索引网格 + 单个艺术家作品全集视图（约 50 位重要艺术家各含数十件代表作）
- **中英双语**：一键切换 ZH / EN，选择记忆于本地存储
- **高清大图灯箱**：点击作品图全屏查看，支持滚轮缩放、双击放大、拖动平移，并可跳转 Wikimedia 查看最大原图
- **时间线索引**：按世纪分期的可点击索引条，配合按年代时序排列
- 实时**搜索**（中英文标题 / 艺术家 / 年代 / 流派 / 媒材 / 馆藏地）
- **筛选**：时代、媒材、地区三重联动
- **时代快捷标签**、随机作品、网格 / 列表视图切换、深链分享、回到顶部
- **详情弹窗**：大图 + 完整信息，支持 ← / → 键翻阅、Esc 关闭
- 纯前端，无需构建，无运行时依赖；响应式适配手机 / 平板 / 桌面

## 本地运行

直接用浏览器打开 `index.html` 即可（图片为本地资源，无需联网）。

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构 |
| `style.css` | 深色画廊风格样式 |
| `lang.js` | 中英双语界面文案 |
| `data.js` | 1001 件作品数据集（双语字段、可排序年份、本地图片路径） |
| `app.js` | 搜索 / 筛选 / 分页 / 弹窗 / 灯箱 / 时间线 / 语言切换逻辑 |
| `images/` | 本地缓存的高清图片（`images/t/` 为缩略图） |

## 数据与图片来源

作品元数据采集自 [Wikidata](https://www.wikidata.org/) 结构化数据（按维基站点链接数即"名气"排序），图片来自 [Wikimedia Commons](https://commons.wikimedia.org/)，均已下载到本仓库 `images/` 目录（大图约 860px、缩略图约 340px）。少数 20 世纪受版权保护、无法转载的名作（如《格尔尼卡》《记忆的永恒》）显示按时代配色的优雅占位卡，并提供「在 Wikipedia 查看」链接，确保页面上不出现破损图片。

> 构建脚本（`_` 前缀）用于离线采集与图片下载，已通过 `.gitignore` 排除，不影响站点运行。
