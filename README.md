# 1001 件人类艺术瑰宝

一个精选并展示人类历史上 1001 件著名艺术作品的静态网站，跨越史前壁画到当代艺术。

## 在线预览

部署到 GitHub Pages 后访问：`https://<你的用户名>.github.io/<仓库名>/`

## 功能

- **1001 件作品**，其中 160+ 件世界名作配有 Wikimedia Commons 真实图片
- **中英双语**：一键切换 ZH / EN，选择记忆于本地存储
- **高清大图灯箱**：点击作品图全屏查看，支持滚轮缩放、双击放大、拖动平移，并可跳转 Wikimedia 查看最大原图
- **时间线索引**：按世纪分期的可点击索引条，配合按年代时序排列
- 实时**搜索**（中英文标题 / 艺术家 / 年代 / 流派 / 媒材 / 馆藏地）
- **筛选**：时代、媒材、地区三重联动
- **时代快捷标签**、随机作品、网格 / 列表视图切换
- **详情弹窗**：大图 + 完整信息，支持 ← / → 键翻阅、Esc 关闭
- 纯前端，无需构建，无依赖；响应式适配手机 / 平板 / 桌面

## 本地运行

直接用浏览器打开 `index.html` 即可。

## 文件结构

| 文件 | 说明 |
|------|------|
| `index.html` | 页面结构 |
| `style.css` | 深色画廊风格样式 |
| `lang.js` | 中英双语资源：界面文案、翻译字典、作品英文标题与说明 |
| `data.js` | 1001 件作品数据集（含双语字段与可排序年份） |
| `app.js` | 搜索 / 筛选 / 分页 / 弹窗 / 灯箱 / 时间线 / 语言切换逻辑 |

## 说明

精选名作的图片来自 [Wikimedia Commons](https://commons.wikimedia.org/) 公共资源，需联网显示；其余条目使用按时代配色的占位卡。
